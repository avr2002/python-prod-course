# `git`


>**Concepts:**
- repositories
- commits
- branches
- merging
- tags
  
* **

> **Commands:**

- `git init`
- `git config`
  - `git config --global user.name "Amit Vikram Raj"`
  - `git config --global user.email "xyz@email.com`
- `git add <file>`
- `git commit -m "message"`
- `git pull [origin xxx]`
- `git push [origin xxx]`
- `git branch`
- `git branch -d`
- `git checkout` or `git switch`
- `git checkout -b <branch_name>`
- `git log`
- `git merge`
- `git rebase`
- `git diff`
- `git tag`
- `git stash`
- `git stash list`
- `git stash pop` or `git stash apply`
- `git reset`
- `HEAD`
